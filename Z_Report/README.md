# سامانه مدیریت و یکپارچه‌سازی فایل‌ها

یک سامانه تحت وب برای نمایش ساختار فایل‌ها و درایوها و یکپارچه‌سازی فایل‌های انتخاب شده.

## ویژگی‌ها

1. **نمایش ساختار درختی** درایوها و فایل‌ها در سمت راست
2. **آیکون‌های رنگی** برای انواع مختلف فایل‌ها و پوشه‌ها
3. **انتخاب چندگانه** فایل‌ها و پوشه‌ها
4. **جستجوی پیشرفته** در ساختار فایل‌ها
5. **یکپارچه‌سازی** فایل‌های انتخاب شده در یک فایل
6. **دانلود خودکار** فایل خروجی
7. **واکنش‌گرا** (Responsive Design)
8. **پشتیبانی از فارسی**

## نصب و راه‌اندازی

1. تمام فایل‌ها را در یک پوشه قرار دهید
2. مطمئن شوید PHP نصب شده است
3. مرورگر را باز کنید و به آدرس `http://localhost/path/to/files/` بروید

## نکات امنیتی

- سیستم فقط فایل‌های با پسوندهای مجاز را نمایش می‌دهد
- محدودیت حجمی برای فایل‌ها اعمال شده است
- فایل‌های خروجی پس از دانلود حذف می‌شوند
- دسترسی به فایل‌های حساس محدود شده است

## سیستم مورد نیاز

- PHP 7.4 یا بالاتر
- Apache با ماژول mod_rewrite
- دسترسی خواندن به فایل‌های سیستم
